JP Math 0.1.0 (powered by Apache)
=================================

The JP Math implements the XorShift1024* prng ([Vigna](http://arxiv.org/abs/1402.6246)) as an extension of [Apache Commons](http://commons.apache.org/) (tm) Math classes.
